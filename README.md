# 🗺️ Your Cairo Trip

A professional tourism and trip booking application for Cairo, featuring a high-performance Flutter mobile client and a robust Laravel RESTful API.

# 🌟 Key Features:
* Explore Cairo: Browse historical and modern landmarks with detailed descriptions, pricing, and working hours.
* Trip Booking: seamless booking system allowing users to schedule visits, specify person counts, and track trip status.
* Favorites System: Save preferred locations to a personalized wishlist for easy access later.
* Ratings & Reviews: View average ratings and share feedback on completed trips with custom Arabic labels
* Personal Profile: Track your trip history, manage favorites count, and update personal information.
* Support Center: Integrated support ticket system to contact administrators directly.
* Secure Authentication: Protected user accounts using Laravel Sanctum tokens.

# 🔧 Tech Stack & Packages:
Frontend (Flutter)
* Dio & Retrofit: Advanced API communication with the Laravel backend.
* Multi-language Support:** Full support for **Arabic and English** using the `easy_localization` package, providing a seamless experience for both local and international tourists.
* Flutter Bloc (Cubit): State management for a predictable and reactive UI.
* GetIt: Dependency injection for clean and modular service management.
* Flutter ScreenUtil: Ensures a perfectly responsive design across all screen sizes.
* Cached Network Image: Optimized performance for loading Cairo's beautiful landmark photos.
Backend (Laravel)
* RESTful API: Structured and documented endpoints for all mobile operations.
* Sanctum: Secure token-based authentication.
* MySQL Database: Optimized schema with foreign keys and indexes for fast querying.
* Image Intervention: Server-side image handling and storage management.
